# Smart Bill Calculator

A small Python console app that calculates discount, VAT, tip, and final total for a purchase.

## About

This project was built to practice Python fundamentals through a simple real-world example.

It includes:

- discount calculation
- VAT calculation
- tip calculation
- final bill summary
- basic input validation

## Concepts Practiced

- variables
- functions
- conditionals
- return values
- arithmetic operations
- formatted output

## Inputs

The program asks for:

- base price
- discount percentage
- VAT percentage
- tip percentage

## Calculation Flow

1. Apply discount
2. Apply VAT
3. Apply tip
4. Show final total

## Run

```bash
python app.py
```
